1. HTML & CSS

HTML & CSS sebetulnya bukanlah sebuah bahasa pemrograman itu sendiri. Lebih seperti wadah atau media untuk perancangan sebuah website dengan menggunakan markup. Di dalam HTML terdapat beragam jenis tags, properties, attributes dan sebagainya. Setiap tag di dalam HTML ditulisan berada di dalam angle brackets seperti tag <\header>. Ketika ditampilkan di dalam browser, tag <\header> tidak akan ditampilkan lagi. CSS juga perlakukannya sama seperti HTML, namun CSS dikhususkan untuk melakukan styling pada dokumen html. Cara untuk menggabungkannya, pada bagian html diberi tambahan tag <\link> dan diberi atribut href yang diikuti dengan nama file .css

Beberapa tag yang biasa dipakai dalam HTML adalah sebagai berikut:

  • html
  • head
  • body
  • header
  • h1, h2, h3,...
  • p
  • div
  • footer
  • ul
  • ol
Sedangkan beberapa tag yang biasa dipakai dalam CSS adalah sebagai berikut:
  • *(universal tag)
  • #(id tag)
  • .(class tag)
Biasanya untuk ID dan class diikuti dengan tag tujuan yang akan diberi style sedangkan untuk (*) karena sudah universal jadi memengaruhi seluruh elemen HTML
